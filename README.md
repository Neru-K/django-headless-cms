Django学習中です。ヘッドレスCMSっぽいのを作りたいですが、まだ何もできてません(^-^;)
初学者ゆえ至らないところ多いかと思いますが、ご指導いただければ大変嬉しいです。

REST Framework使ってます。
/__admin　で管理画面ログイン
/api/v1/dj-rest-auth/registration　でアカウント追加
/api/v1 で記事投稿&一覧確認
/api/v1/1　←後ろに番号付けたら各記事編集
アカウント作成＆記事の追加や編集は、管理画面にログインして行うことももちろんできます。
